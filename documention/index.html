Getting Started

Installation
  • Node.js and npm : You can download Node.js from https://nodejs.org. npx comes bundled with Node.js.
Install NPX Packages & Yarn
  • npm : You must have to npm install packages. You can do this by running npx install from the root of your project to install all the necessary dependencies.
Development server
  • Run yarn start for a dev server. Navigate to http://localhost:3000/ . The app will automatically reload if you change any of the source files.
    • or
  • Run npm start for a dev server. Navigate to http://localhost:3000/ . The app will automatically reload if you change any of the source files.
Build
  • Run yarn build to build the project. The build artifacts will be stored in the build/ directory. Use the yarn build flag for a production build.
    • or
  • Run npm run build to build the project. The build artifacts will be stored in the build/ directory. Use the npm run build flag for a production build.
If you want to run these template on React Js server
After downloading the file from themeforest, You will find dustrix.zip file.Then unzip the dustrix.zip and run the following commands
  1. yarn on dustrix folder.
  2. yarn web dustrix folder.
  3. Please go to address localhost:3000 on your browser and You will find dustrix landing
  4. Learn (A tool for managing JavaScript projects with multiple packages. https://lernajs.io )
  5. React Js & JSX https://reactjs.org/docs/getting-started.html
  6. Styled System and Styled Components
  7. Edit with Your - Done
  8. More Learn - https://reactjs.org/tutorial/tutorial.html
